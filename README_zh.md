# 特性
- 💪 实时卡牌部件切换 --- 不使用预导出的完整卡牌框架，而将部件拆分并使用菜单实时切换。
- 🌏 i18n（国际化） --- 多语言支持。
- 👀 保持最新 --- 和最新版炉石传说内容保持同步。
- 😁 高相似度 --- 卡牌样式尽量还原游戏中的样式。
- **PWA**(渐进式Web应用) --- 支持在桌面/移动端离线使用。
